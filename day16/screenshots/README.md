## Screenshot Tips

### Quality Requirements:
- **Resolution:** Minimum 1920x1080 (Full HD)
- **Format:** PNG (lossless)
- **Visibility:** All text should be readable without zooming
- **Context:** Include chart titles and axis labels

### Metabase Screenshot Tips:
1. **Full Screen Mode:** Click the expand icon (‚õ∂) on each card for cleaner capture
2. **Zoom Level:** Adjust browser zoom (Cmd/Ctrl + or -) to fit content
3. **Dark Mode:** Consider using light mode for better contrast (Settings ‚Üí Account ‚Üí Appearance)
4. **Hide UI:** Hide Metabase UI elements if possible (focus on viz content)

### Capture Methods:

**macOS:**
```bash
# Full screen
Cmd + Shift + 3

# Selected area (recommended)
Cmd + Shift + 4
# Then drag to select the card area
```

**Windows:**
```bash
# Snipping Tool
Win + Shift + S

# Or use built-in screenshot (Win + PrtScn)
```

**Linux:**
```bash
# gnome-screenshot
gnome-screenshot -a

# Or use Shutter, Flameshot, etc.
```
